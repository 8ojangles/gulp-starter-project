Colour Palette

The full list of site colors are set in a JSON file and compiled into a SCSS/SASS map. This is then run through a function to create al lthe color utility classes for background color and text color. in the case of the background color, an additional color attribute is set for contrasting text color to make accessible contrasts automatically applied to child elelements. The median color (level 400) should be regarded as the base (brand) value for each color, above and below shades should be provided for situations where variations are required.
