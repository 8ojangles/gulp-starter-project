easingFn

Base function signature for easing methods. {currentIteration} - the current clock or iteration cycle {startValue} - The value to EASE from {changeInValue} - The change value relative to the start value {totalIterations} -The total cycles or iterations of the easing curve to calculate {returns} - The calculated (absolute) value along the easing curve from the {startValue}

easingFn(currentIteration: number, startValue: number, changeInValue: number, totalIterations: number): number

Type: Function

Parameters
currentIteration (number) The current iteration as a proportion of the totalIteration parameter.
startValue (number) The starting value
changeInValue (number) The change value relative to the start value.
totalIterations (number) The total iterations of the easing curve to calculate.
Returns
number: The calculated (absolute) value along the easing curve

easingFnExtended

Base function signature for easing methods. {currentIteration} - the current clock or iteration cycle {startValue} - The value to EASE from {changeInValue} - The change value relative to the start value {totalIterations} -The total cycles or iterations of the easing curve to calculate {overshoot} - a ratio of the {startValue} and {changeInValue} to give the effect of "overshooting" the initial {startValue} (easeIn), "overshooting" final value {startValue + changeInValue} (easeOut) or both (easeInOut*). {returns} - The calculated (absolute) value along the easing curve from the {startValue}
